NUTRICLOCK PWA

Este pacote está pronto para publicação em hospedagem estática.

Forma mais simples:
1. Acesse app.netlify.com/drop em um computador.
2. Arraste a pasta NutriClock_PWA ou este ZIP para a página.
3. A Netlify fornecerá um endereço HTTPS.
4. Abra o endereço no Safari do iPhone.
5. Toque em Compartilhar > Adicionar à Tela de Início.

Também funciona no GitHub Pages, Cloudflare Pages e Vercel.

Importante: os registros ficam no navegador do aparelho. Use Exportar JSON para backup.
