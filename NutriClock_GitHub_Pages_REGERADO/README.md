# NutriClock

Aplicativo web instalável para acompanhar calorias, exercícios, água, cafeína, proteína e peso.

## Publicação no GitHub Pages

1. Abra o repositório: https://github.com/guirart/nutriclock
2. Clique em **Add file**.
3. Clique em **Upload files**.
4. Extraia este arquivo ZIP no computador.
5. Envie todos os arquivos extraídos para a raiz do repositório.
6. Clique em **Commit changes**.
7. Vá em **Settings → Pages**.
8. Em **Source**, selecione **Deploy from a branch**.
9. Escolha a branch **main** e a pasta **/(root)**.
10. Clique em **Save**.

O site ficará disponível em:

https://guirart.github.io/nutriclock/

## Instalação no iPhone

Abra o link no Safari e toque em:

**Compartilhar → Adicionar à Tela de Início**

## Privacidade

Os dados são salvos no navegador do aparelho por LocalStorage.
Use a opção de exportar JSON para manter um backup.
