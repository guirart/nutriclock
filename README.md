# NutriClock v7.2 — Baús, drops e inventário

Novidades:
- baús guardados automaticamente ao completar água, proteína, treino e equilíbrio calórico;
- raridades Normal, Raro, Ultrararo, Épico, Lendário e Mítico;
- probabilidades decrescentes;
- drops aleatórios;
- inventário persistente no navegador;
- seis espaços de equipamento;
- atributos diferentes por item;
- itens equipados aparecem junto do primata;
- modal de revelação de loot.
