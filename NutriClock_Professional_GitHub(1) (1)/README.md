# NutriClock Professional

Aplicativo web instalável para acompanhar:

- calorias
- exercício
- saldo líquido
- proteína
- água
- cafeína
- peso
- histórico diário e semanal
- backup em JSON

## Publicar no GitHub

Repositório:

https://github.com/guirart/nutriclock

### Pelo site do GitHub

1. Extraia o ZIP.
2. Abra o repositório.
3. Clique em **uploading an existing file**.
4. Arraste todos os arquivos e pastas extraídos.
5. Clique em **Commit changes**.
6. Vá em **Settings → Pages**.
7. Em **Source**, selecione **GitHub Actions**.
8. Aguarde a publicação.

O endereço será:

https://guirart.github.io/nutriclock/

## Instalar no iPhone

1. Abra o endereço no Safari.
2. Toque em **Compartilhar**.
3. Toque em **Adicionar à Tela de Início**.

## Privacidade

Os dados ficam salvos no navegador do aparelho usando `localStorage`.

Use **Exportar JSON** para fazer backup.
